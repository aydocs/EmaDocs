- **Variant**: neo

## Usage

### HTML Structure
```html
<div class="ema-paginate ema-paginate--neo">
    Paginate neo
</div>
```

### With Props
```html
<div class="ema-paginate ema-paginate--neo" 
     data-size="large" 
     data-disabled="false">
    Paginate neo (Enhanced)
</div>
```

### Interactive
```html
<div class="ema-paginate ema-paginate--neo" 
     onclick="handleClick()">
    Click me!
</div>
```
