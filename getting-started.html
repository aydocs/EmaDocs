🚀 Adiox Framework Nasıl Çalışır?

Adiox'u kullanmaya başlamak için sadece 3 adım!

1️⃣ Dosyaları Dahil Et

HTML dosyanıza CSS ve JS dosyalarını ekleyin:

<!-- CSS -->
<link rel="stylesheet" href="css/adiox.css">

<!-- JavaScript (body sonunda) -->
<script src="js/adiox.js" type="module"></script>

2️⃣ Bileşenleri Kullan

Adiox bileşenlerini HTML'de direkt kullanabilirsiniz:

Buton Örneği:

Birincil Buton Başarılı Tehlike 
<adi-button variant="primary">Birincil Buton</adi-button>
<adi-button variant="success">Başarılı</adi-button>
<adi-button variant="danger">Tehlike</adi-button>

Kart Örneği:

Kart Başlığı

Bu bir kart içeriğidir. Adiox kartları header, body ve footer slotlarına sahiptir.

Kaydet 
<adi-card>
  <div slot="header">Kart Başlığı</div>
  <div slot="body">İçerik buraya...</div>
  <div slot="footer">
    <adi-button variant="primary">Kaydet</adi-button>
  </div>
</adi-card>

3️⃣ JavaScript API'yi Kullan

Toast Bildirimleri:

Başarılı Toast Hata Toast Bilgi Toast 
// Toast göster
Adiox.UI.toast('İşlem başarılı!', 'success');
Adiox.UI.toast('Bir hata oluştu!', 'error');
Adiox.UI.toast('Bilgi mesajı', 'info');

Modal Diyaloglar:

Modal Aç 
// Modal göster
Adiox.UI.modal({
  title: 'Onay Gerekli',
  content: 'Bu işlemi yapmak istediğinizden emin misiniz?',
  buttons: [
    { text: 'İptal', variant: 'outline' },
    { text: 'Onayla', variant: 'primary' }
  ]
}).then(result => {
  console.log('Seçilen:', result);
});

Reaktif Store:

Sayaç: 0

Artır Azalt 
// Store oluştur
const store = Adiox.Store.create({
  count: 0
});

// Store'u HTML'e bağla
store.bind('count', '#counter');

// Store'u güncelle
document.querySelector('#incrementBtn').addEventListener('click', () => {
  store.set('count', store.get('count') + 1);
});

// Store değişikliklerini izle
store.watch('count', (newValue, oldValue) => {
  console.log(`Sayaç ${oldValue}'dan ${newValue}'a değişti`);
});

SPA Router:

Ana Sayfa Hakkında 
// Route tanımla
Adiox.Router.addRoute('/', () => {
  document.querySelector('#app').innerHTML = '<h1>Ana Sayfa</h1>';
});

Adiox.Router.addRoute('/about', () => {
  document.querySelector('#app').innerHTML = '<h1>Hakkında</h1>';
});

// Router'ı başlat
Adiox.Router.init();

// Programatik navigasyon
Adiox.Router.navigate('/about');

Animasyonlar:

Animasyon Kutusu
Fade In Slide Up Bounce 
// Animasyon uygula
Adiox.Animations.animate('#animBox', 'fadeIn', { duration: 500 });
Adiox.Animations.animate('#animBox', 'slideUp', { duration: 600 });
Adiox.Animations.animate('#animBox', 'bounce', { duration: 800 });

🎨 Utility Classes (Tailwind Tarzı)

Adiox, Tailwind benzeri utility class'lar sunar:

Spacing:

p-4, m-2, px-6, py-3, mt-8, mb-4

Colors:

bg-primary, text-danger, border-success

Typography:

text-xl, font-bold, text-center

Layout:

flex, grid, items-center, justify-between 
<div class="flex items-center justify-between p-4 bg-primary text-white rounded">
  <h3 class="text-xl font-bold">Başlık</h3>
  <adi-button variant="outline">Buton</adi-button>
</div>

🌙 Dark Mode

Adiox otomatik dark mode desteği sunar:

Temayı Değiştir 
// Temayı değiştir
Adiox.Theme.setTheme('dark');  // veya 'light'
Adiox.Theme.toggle();          // Tema değiştir

// Mevcut temayı al
const currentTheme = Adiox.Theme.getTheme();
