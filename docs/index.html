Getting Started

Get up and running with emadocs in minutes.

Installation

Include the framework files in your HTML:

<!DOCTYPE html>
<html lang="en">
<head>
  <link rel="stylesheet" href="css/emadocs.css">
</head>
<body>
  <div id="app"></div>

  <script src="js/core.js"></script>
  <script src="js/emadocs.js"></script>
</body>
</html>

Basic Usage

// Initialize the framework
Emadocs.init({
  router: { mode: 'hash' }
});

// Show a notification
Emadocs.UI.toast('Welcome to emadocs!', { 
  type: 'success' 
});

// Store data
Emadocs.Store.set('user', { name: 'John' });

// Navigate
Emadocs.Router.navigate('/dashboard');

3D Engine

Built-in WebGL 3D engine with physics simulation.

Creating a 3D Scene

// Initialize 3D canvas
const scene = Ema3D.init('myCanvas');

// Create objects
Ema3D.createCube('myCanvas', {
  color: [0.54, 0.36, 0.96, 1.0],
  rotationSpeed: { x: 0.01, y: 0.02, z: 0 }
});

Ema3D.createSphere('myCanvas', {
  position: { x: 2, y: 0, z: 0 }
});

// Start rendering
Ema3D.start('myCanvas');

3D Features

  • Multiple 3D shapes (cube, sphere, torus, cylinder)
  • Particle systems with 1000+ particles
  • Physics simulation with gravity and collisions
  • Camera controls and animations
  • Post-processing effects (bloom, motion blur)

Theming

emadocs features a beautiful purple theme with full customization.

Color System

:root {
  --ema-primary: #8b5cf6;
  --ema-accent: #ec4899;
  --ema-success: #10b981;
  --ema-warning: #f59e0b;
  --ema-danger: #ef4444;
}

Dark Mode

// Toggle theme
Emadocs.Theme.toggle();

// Set specific theme
Emadocs.Theme.set('dark');

// Check current theme
if (Emadocs.Theme.isDark()) {
  console.log('Dark mode active');
}

API Reference

Complete API documentation for all modules.

Router API

// Add route
Emadocs.Router.addRoute('/path', () => {
  return '<div>Content</div>';
});

// Navigate
Emadocs.Router.navigate('/path');

// Get current route
const route = Emadocs.Router.getCurrentRoute();

Store API

// Set value
Emadocs.Store.set('key', value);

// Get value
const value = Emadocs.Store.get('key');

// Watch changes
Emadocs.Store.watch('key', (newVal, oldVal) => {
  console.log('Changed:', newVal);
});

// Bind to DOM
Emadocs.Store.bind('key', '#element');

UI API

// Toast
Emadocs.UI.toast('Message', { type: 'success' });

// Modal
Emadocs.UI.modal({
  title: 'Title',
  html: '<p>Content</p>',
  primaryText: 'OK'
});

// Alert
await Emadocs.UI.alert('Alert message');

// Confirm
const confirmed = await Emadocs.UI.confirm('Are you sure?');
