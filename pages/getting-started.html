Download & Include

Download the framework files and include them directly in your HTML. Perfect for quick prototyping and simple projects.

HTML
<link rel="stylesheet" href="css/ema-colors.css">
<link rel="stylesheet" href="css/emadocs.css">
<script src="js/core.js"></script>
<script src="js/app-init.js"></script>
🌐

CDN (Coming Soon)

Include Ema Framework directly from our CDN. No download required, always get the latest version automatically.

HTML
<link rel="stylesheet" href="https://cdn.ema.dev/latest/ema.css">
<script src="https://cdn.ema.dev/latest/ema.js"></script>
1

Initialize Ema Framework

Create a new Ema instance and initialize it with your preferred settings. You can customize the theme, enable 3D effects, and configure other options.

JavaScript
const app = new Ema({
  'theme': 'purple',
  '3d': true,
  'animations': true
});
app.init();
2

Create Your First Component

Use Ema's component system to create beautiful UI elements. Each component comes with 8 color variants and advanced customization options.

JavaScript
// Create a button component
const button = app.createComponent('button', {
  'text': 'Hello Ema!',
  'variant': 'purple',
  'size': 'large'
});
// Add to your page
document.getElementById('app').appendChild(button.element);
3

Add 3D Effects & Animations

Enhance your application with stunning 3D effects and smooth animations. Ema's 3D engine is powered by WebGL for optimal performance.

JavaScript
// Add 3D effects on button click
button.onClick(() => {
  app.effects.createExplosion({
    'x': button.x,
    'y': button.y,
    'color': [0.4, 0.2, 0.7]
  });
});
4

Build Your Application

Combine multiple components to create complex applications. Use Ema's advanced features like routing, state management, and real-time updates.

JavaScript
// Create a complete dashboard
const dashboard = app.createComponent('dashboard', {
  'charts': true,
  'realTime': true,
  'theme': 'purple'
});
