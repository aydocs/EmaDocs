Core API

Essential methods and utilities for initializing and managing your Ema Framework application.

Constructor new EmaApp(options)

Initialize a new Ema Framework application with configuration options.

Parameter Type Required Description
theme string No Theme name ('dark', 'light', 'auto')
animations boolean No Enable/disable animations
components array No Array of component names to preload
terminal â€” bash â€” 80x24
// Initialize Ema Framework
const app = new EmaApp({
    theme: 'dark',
    animations: true,
    components: ['button', 'form', 'chart', 'modal']
});

// Access global instance
console.log(app.version); // "2.0.0"
console.log(app.theme);   // "dark"
Method app.createComponent(type, options)

Create a new component instance with specified options.

JavaScript
terminal â€” bash â€” 80x24
// Create a button component
const button = app.createComponent('button', {
    text: 'Click Me',
    variant: 'primary',
    size: 'large',
    onClick: () => {
        console.log('Button clicked!');
    }
});

// Add to DOM
document.body.appendChild(button.element);

// Update component
button.update({ text: 'Updated!' });

Components API

Detailed documentation for all available UI components and their methods.

Component Button Component

Interactive button component with multiple variants and states.

Property Type Default Description
text string '' Button text content
variant string 'primary' Color variant (primary, secondary, success, etc.)
size string 'medium' Button size (small, medium, large)
disabled boolean false Disable button interaction
onClick function null Click event handler
JavaScript 
// Create different button variants
const primaryBtn = app.createComponent('button', {
    text: 'Primary',
    variant: 'primary',
    onClick: () => alert('Primary clicked!')
});

const successBtn = app.createComponent('button', {
    text: 'Success',
    variant: 'success',
    size: 'large',
    onClick: () => alert('Success clicked!')
});

const dangerBtn = app.createComponent('button', {
    text: 'Danger',
    variant: 'danger',
    disabled: true
});
