‚ö° Core Framework

The foundation of Ema Framework. Core initialization, configuration, and essential functionality for building modern web applications.

Constructor

new Ema(options)

Creates a new Ema Framework instance with the specified configuration options.

Parameters

Name Type Required Description
options Object No Configuration options for the framework
options.theme String No Theme name (purple, blue, green, etc.)
options.3d Boolean No Enable 3D engine features
Basic Ema Initialization
const app = new Ema();
// Initialize with default settings
app.init();
Advanced Configuration
const app = new Ema({
  'theme': 'purple',
  '3d': true,
  'animations': true,
  'debug': false
});
Method

init()

Initializes the Ema Framework with all configured options and sets up the application.

Usage
const app = new Ema();
app.init();
// Framework is now ready to use

Method

createComponent(type, options)

Creates a new component instance with the specified type and options.

Parameters

Name Type Required Description
type String Yes Component type (button, card, input, etc.)
options Object No Component configuration options
Example

  'text': 'Click Me',
  'variant': 'purple',
  'size': 'large'
});
